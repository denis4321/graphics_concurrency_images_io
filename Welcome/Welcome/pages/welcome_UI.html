The JBuilder development environment

Features vary by JBuilder edition

The JBuilder integrated development environment (IDE) consists of a window that contains several panes and panels for performing most development functions: editing, visual designing, navigating, browsing, comparing, modeling, building, compiling, debugging, running, optimizing, and auditing.

Note: To expand the content pane to full width (hiding the project and structure panes), choose View|Maximize Content Pane from JBuilder's main menu.

Use the Preferences dialog box (Tools|Preferences) to customize the development environment. You can control such IDE preferences as:

  • Browser settings, including build, content pane, status bar, file types, and audio options
  • Fonts options for the editor, browser, and output panes
  • Keymaps customization
  • Editor display, actions, color, and settings for CodeInsight, TagInsight, and code templates
  • Markup language editing and formatting (JBuilder Developer and Enterprise)
  • Structure pane and icon display for Java file sorting and filtering
  • Web run, debug, and optimize settings (JBuilder Developer and Enterprise)
  • XML, including catalog settings (JBuilder Developer and Enterprise)
  • UML display (JBuilder Enterprise)
  • EJB designer (JBuilder Enterprise)
  • Help settings
  • Mobile Screen (MIDP only) (JBuilder Developer and Enterprise)
