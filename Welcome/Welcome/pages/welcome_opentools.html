OpenTools

JBuilder's OpenTools simplify customizing and adding to JBuilder. The OpenTools API reference documentation provides Javadoc-derived documentation for each OpenTool. These expose the programmatic "hooks" you need to customize and extend JBuilder to match your own needs and preferences. Access to OpenTools features varies by JBuilder edition.
